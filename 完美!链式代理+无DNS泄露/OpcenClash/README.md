# 这是关于配置【OpcenClash】的相关配置文件及说明的目录！

---

**`自动订阅+链式代理.yaml`   ------------------ OpcenClash 目录下的订阅配置文件，用于链式代理+机场聚合+自愈故障转移的配置文件；在" 配置管理"页面编辑上传切换配置运行会自动同步机场节点。不建议在“配置订阅”页面中添加自动自动订阅。**

**``openclash`` ------------------ OpcenClash 目录下的openclash主程序配置文件为防止DNS泄露的最佳配置文件只需要上传到``/etc/config/``目录重启openclash即可；需要关闭openclash系统下的“网络”--->“DHCP/DNS”--->“DNS 重定向”选项**
